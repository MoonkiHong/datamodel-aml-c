AML_EXPORT CAMLErrorCode CreateRepresentation ( const char *  filePath,
representation_t *  repHandle 
)

Create an instance of Representation.

Parameters
filePath[in] path of an AML file that contains data model information.
repHandle[out] handle of created Representation.
Return values
#CAML_OKSuccessful.
#CAML_INVALID_PARAMInvalid parameter.
#CAML_INVALID_FILE_PATHInvalid file path.
#CAML_NO_MEMORYFailed to alloc memory.
Note
Representation instance will be allocated, so it should be deleted after use. To destroy an instance, use DestroyRepresentation().
